Coffe Shop Map
Tableu dashboard

I built this project as a huge coffee fanatic looking for a tool that could help me navigate the vast array of beautiful coffee shops in Austin, Tx. Using Yelp's API and Python for data cleaning I created a Tableu dashboard that maps all coffee shops in the city based on their price and reviews.
